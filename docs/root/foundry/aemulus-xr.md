# Aemulus-XR

**Real-time Logic & State Management for Extended Reality**

Lightweight event-driven state machine designed for low-latency XR applications. Replaces bloated middleware with direct hardware integration.

## Overview
Aemulus handles sensor fusion, gesture recognition, and spatial mapping in constrained compute environments. Proves that enterprise XR doesn't require thick abstractions.

## Status
**OPERATIONAL** — Deployed in field testing scenarios.

## Core Features
- Event-driven state machine
- Gesture recognition pipeline
- Sensor fusion (IMU + camera)
- Spatial anchor management
